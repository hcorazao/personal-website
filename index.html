Project Bambu - Scotiabank
Previous Next

Local bank web application developed completely from scratch using Angular 5 and Angular Material for the UI. It's a very complex 7 steps form with 5 different types of dynamic input components. It has a very cool animation developed in pure css that fades out the last input question and focus the next one.

There is a pinpad component that is used by clients to create the bank private password to use in transactions. It has maximum security using double encryption and 3 layer of security (frontend, backend, core).

There is also a confetti component in the last step that appears when the user finishes the process and creates the bank account.
